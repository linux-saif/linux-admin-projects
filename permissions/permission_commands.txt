This project demonstrates basic Linux file and directory permissions, ownership management, and access control using standard Linux commands.
It is designed for beginners and junior Linux administrators to understand how permissions work in real-world environments.
# Prerequisites

    Linux system (Ubuntu / CentOS / RHEL / Fedora)

    Basic command-line knowledge

    Non-root user with sudo access
